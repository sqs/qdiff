# qdiff

A terminal user interface for viewing Git diffs, staging/unstaging, reverting, and fixup'ing, inspired by [Magit](https://magit.vc/).

## Keyboard Shortcuts

| Key | Action |
| --- | --- |
| `ArrowDown` | Navigate to next file |
| `ArrowUp` | Navigate to previous file |
| `s` | Stage file |
| `u` | Unstage file |
| `Tab` | Toggle expand/collapse diff |
| `q` | Quit |
